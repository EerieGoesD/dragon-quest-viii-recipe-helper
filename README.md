# Dragon Quest VIII Tracker (Alchemy, Inventory, Craftable, Optimizer)

## Overview
The **Dragon Quest VIII Tracker** is a browser-based tool that combines an **item/inventory tracker**, an **alchemy recipe tracker**, a **craftable-recipe calculator**, and an **equipment optimizer** for *Dragon Quest VIII*. As you tick the items and equipment you own, the tool updates which recipes are craftable, tracks which recipes youâ€™ve created, and can calculate recommended equipment based on the gear currently marked in your inventory. Progress is saved locally in the browser, with optional JSON export/import.

## Website

## Features
- **Item + equipment inventory tracker** (checkbox list)
- **Alchemy recipe tracker** with created-state tracking
- **Craftable tracker**:
  - Shows recipes craftable from the currently selected inventory
  - Shows craftable recipes not yet marked as created
- **Recipe filters** (equipment vs items)
- **Progress saving**:
  - Automatic local saving in the current browser
  - JSON export/import
- **Equipment optimizer**:
  - Selects the highest **Attack** weapon and highest **Defence** armour/shield/helmet each character can equip from the tracked inventory
