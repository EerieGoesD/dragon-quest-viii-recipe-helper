# Dragon Quest VIII Alchemy Tracker

## Overview
The **Dragon Quest VIII Alchemy Tracker** is a lightweight, browser-based tool to help Dragon Quest VIII players track **inventory**, **Alchemy Pot recipes**, and overall **crafting progress**. It calculates which recipes are currently craftable based on your inventory ticks and saves your progress locally in your browser (with export/import support).

## Features
- **Inventory tracker** (tick items + equipment you own)
- **All recipes table** with:
  - Category
  - Result item
  - Ingredients
  - Craftable status
  - Created tracking
- Filters for **Equipment** and **Items**
- Separate sections for:
  - **Craftable Recipes**
  - **Craftable Recipes (Not Created)**
  - **Your Recipe List (Created)**
- **Export/Import progress** (JSON)
- **Equipment Optimizer**:
  - Picks the highest **Attack** weapon and highest **Defence** armour/shield/helmet each character can equip from your inventory
